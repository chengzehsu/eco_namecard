# Sentry DSN 設定範本
# 請將下面的 YOUR_SENTRY_DSN_HERE 替換為你從 Sentry 獲取的實際 DSN

# DSN 格式範例：
# https://xxxxxxxxxxxxxxxx@o123456.ingest.sentry.io/123456

# 設定步驟：
# 1. 從 Sentry Dashboard 複製你的 DSN
# 2. 在 Zeabur 環境變數中設定：
#    變數名稱：SENTRY_DSN
#    變數值：[你的 DSN]

# 範例環境變數設定：
SENTRY_DSN=YOUR_SENTRY_DSN_HERE

# 測試用的臨時 DSN (請替換成真實的)：
# SENTRY_DSN=https://example123@o456789.ingest.sentry.io/987654
